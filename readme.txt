== Description ==

All in one solution to track your visitors and record their screen, makes possible to see how they interact with your website using Google Analytics, Hotjar and Smartlook

== Installation ==

To install this plugin, just follow the steps below:

1 - Download the plugin files from WordPres respository
2 - Upload them into your plugins directory or use the WordPress plugin panel to upload it.
3 - Activate the plugin and you'll see a new menu called "Analyticator"


== Frequently Asked Questions ==

= How do i use the plugin? =

1 - Download the plugin files from WordPres respository<br />
2 - Upload them into your plugins directory or use the WordPress plugin panel to upload it.<br />
3 - Activate the plugin and you'll see a new menu called "Analyticator"<br />
4 - Click on it and select wich features you want to use and save<br />
5 - You'll see new submenus appears on Analyticator menu with the name of the features you choose.<br />
6 - Click on which feature and paste the tracking code into it.
7 - Hit the save button and it's done ;)

= How do i know it is working? =

Visit your website and then open the source code on your browser, look for the "Analyticator" word on the source to find our plugin code on the website footer.
